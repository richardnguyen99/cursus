About Cursus

The open-source, public all-in-one API to find and retrieve university information, globally.

Why Cursus?

Cursus is a free, open-source API that provides information about universities, colleges and their courses globally. The intention of this API is to build a singly-uniformed API that any app can integrate seamlessly and don't need to worry about maintaining their own database of the likes.

History of Cursus

Cursus was created to provide a universal API to query university courses and their information, and that is still the core strength of Cursus. The mission of Cursus is to provide basic, essential information about universities and their courses such as name, location, domain, department, course ID, etc.

Development

Cursus is powered by Flask, a Python microframework and it uses PostgresQL as the database in production. Both are suitable for the purpose of Cursus and are easy to use and maintain.

Cursus is still in development and needs your help to grow. If you're interested, please check it out and make contributions at GitHub.

Support queries

Universities

Cursus provides information about universities and colleges globally. The information includes name, location, domain, departments, founders, etc.

Cursus also allows you to search for universities by a variety of criteria so that you can find the university you're looking for easily.

Courses

Cursus provides information about courses offered by universities and colleges globally. The information includes name, ID, department, etc.
